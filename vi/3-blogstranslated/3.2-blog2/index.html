SAP Cloud ERP và SAP BTP đã có mặt trên AWS Marketplace

By Lynn Jones, Arne Knoeller, and Soumya Sekhar Das – 20 AUG 2025

Bức tranh toàn cảnh về enterprise resource planning (ERP) đang chứng kiến một sự chuyển đổi mang tính cách mạng, hướng tới các giải pháp cloud-native và một cách tiếp cận đơn giản hơn để mua, triển khai và quản lý các giải pháp này với tốc độ và mức độ bảo mật cao hơn.

Ở vị trí tiên phong của sự chuyển đổi này, SAP và Amazon Web Services (AWS) tiếp tục củng cố quan hệ đối tác chiến lược, nhằm giúp các tổ chức ở mọi quy mô đổi mới nhanh hơn và vận hành các quy trình kinh doanh quan trọng trong môi trường cloud.

1. Lịch sử ra mắt trên AWS Marketplace

  • AWS re:Invent 2024: SAP Cloud ERP (trước đây gọi là GROW with SAP) được ra mắt trên AWS Marketplace.
  • SAP Sapphire 2025: SAP và AWS giới thiệu SAP Business Technology Platform (BTP) Services trên Marketplace.
  • SAP ra mắt SAP Business Suite, bao gồm:
    • SAP Cloud ERP (GROW)
    • Private (RISE)
    • Các dịch vụ khác từ SAP

GROW và RISE phát triển từ sản phẩm độc lập thành các guided implementation frameworks hay còn gọi là các “hành trình”, giúp khách hàng triển khai Cloud ERP và các sản phẩm khác trong Business Suite một cách có cấu trúc.

2. Lợi ích khi mua SAP Business Suite và SAP BTP qua AWS Marketplace

2.1. Quy trình mua đơn giản (Simplified Procurement Process)

  • Quản lý tất cả thông qua tài khoản AWS hiện tại: một hóa đơn, một bộ điều khoản, nền tảng quản lý tập trung.
  • Cam kết chi tiêu AWS hiện tại có thể tính vào việc mua SAP, tối đa hóa ngân sách công nghệ.
  • Áp dụng cam kết AWS vào Private Pricing spend commitment (PPA) → đơn giản hóa quản lý ngân sách.
  • Rút ngắn thời gian procurement từ vài tuần xuống còn vài ngày, giúp đội ngũ tập trung vào vận hành doanh nghiệp.

2.2. Hướng dẫn tin cậy qua các đối tác (Trusted Guidance through Partners)

Các đối tác SAP & AWS hỗ trợ triển khai S/4HANA Cloud Edition:

  • Cognitus, Navigator, NTT Data, Syntax, Seidor, Navisite, AnswerThink, Bramasol, Vision33, VistaVu
  • Nhiều đối tác khác sẽ được bổ sung khi chương trình phát triển

3. Lợi ích của SAP Business Suite và SAP BTP

3.1. Triển khai nhanh SAP Cloud ERP

  • Thời gian tạo giá trị nhanh hơn 40-60% so với ERP on-premises.
  • Hỗ trợ rollout và go-live nhanh nhờ quy trình dựng sẵn, công cụ hướng dẫn, phương pháp luận dựa trên best practices của SAP và AWS.
  • Bao gồm:
    • Embedded AI agents cho tự động hóa thông minh
    • Pre-configured industry best practices
    • AWS infrastructure optimization, bao gồm Graviton processors
    • Seamless integration với SAP BTP

3.2. Khả năng tích hợp

  • Mở rộng dễ dàng với hơn 75 dịch vụ SAP BTP trên AWS và hơn 200 dịch vụ AWS.
  • Tận dụng AWS global backbone network → giảm độ trễ, bảo mật cao, cải thiện hiệu năng.
  • Traffic giữ trong mạng AWS → nâng cao security posture, vẫn tuân thủ quy định bảo mật nghiêm ngặt.

3.3. Mở rộng với low/no-code tools

  • Khách hàng tận dụng:
    • 750+ prebuilt integrations
    • 700+ open APIs
    • 6000+ reusable CDS views
  • Xây dựng extension tùy chỉnh không cần nhiều effort về coding hay integration phức tạp.

3.4. Hiện đại hóa với AWS & SAP GenAI

  • SAP Cloud ERP giúp đơn giản hóa vận hành kinh doanh, phục vụ người dùng tốt hơn.
  • Xây dựng ứng dụng generative AI an toàn kết hợp với Amazon Bedrock (có trên SAP generative AI hub).
  • Hỗ trợ Joule, generative AI copilot của SAP, hoạt động trên toàn bộ ứng dụng SAP.

3.5. Khả năng mở rộng & hiệu năng

  • Hỗ trợ scalability, high availability và security.
  • Triển khai bản địa hóa theo nhu cầu từng khu vực.
  • Tận dụng đổi mới từ AWS → best-in-class CPU price performance và tiết kiệm năng lượng.

4. Cách đổi mới với SAP Cloud ERP, BTP & AWS

  • SAP BTP cho phép tạo side-by-side extensions mà vẫn giữ clean core của SAP S/4HANA.
  • Developer có thể chọn ngôn ngữ: ABAP, Java, Node.js, Python.
  • Cloud Foundry: hỗ trợ Java, Node.js, Python
  • Kyma: serverless functions (Node.js, Python), microservices cross-platform
  • ABAP Cloud sử dụng RAP, Java với Spring Boot, JavaScript với CAP
  • Tối đa hóa nguồn lực, chọn tech stack phù hợp cho từng use case.

5. Use Cases – Ví dụ đổi mới

5.1. Intelligent Document Processing

  • Amazon Textract tự động trích xuất dữ liệu từ hóa đơn (Accounts Payable).
  • Kết hợp Amazon Comprehend → nhận diện entity, phân tích sentiment.
  • Dữ liệu trả về S/4HANA qua OData APIs.
  • Triển khai như extension trên SAP BTP, tích hợp liền mạch workflow SAP.

5.2. Data-to-Value – Advanced Analytics

  • Xuất dữ liệu từ S/4HANA sang Amazon Redshift cho phân tích nâng cao.
  • Dùng AWS Glue để ETL, đồng bộ real-time.
  • Java developer → Spring Boot, ABAP developer → AWS SDK for ABAP.

6. Lợi ích tổng thể

  • Tính linh hoạt kỹ thuật: tạo, triển khai chức năng mới không ảnh hưởng core system.
  • Cloud-native → scalability tự nhiên, mở rộng từng phần.
  • Dễ bảo trì, tuân thủ clean core → nâng cấp S/4HANA mượt mà.
  • Chi phí pay-per-use cho BTP & AWS.
  • Tập trung vào giá trị kinh doanh, giảm technical debt.

7. Tóm tắt

Việc SAP Cloud ERP Business Suite và SAP BTP có mặt trên AWS Marketplace đánh dấu cột mốc quan trọng trong lĩnh vực giải pháp đám mây doanh nghiệp:

  • Kết hợp khả năng ERP hàng đầu của SAP với hạ tầng mạnh mẽ của AWS.
  • Hơn 200 dịch vụ AWS, LLMs qua Amazon Bedrock.
  • Giải pháp toàn diện, an toàn, khả năng mở rộng và sáng tạo.
  • Đơn giản hóa quá trình mua sắm, quản trị và tăng tốc chuyển đổi số.

Khuyến nghị: liên hệ với AWS và các AWS for SAP Partners để đánh giá IT landscape và xác định nơi SAP Cloud ERP phù hợp nhất.
